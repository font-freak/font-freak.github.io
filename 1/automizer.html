Font Automizer (runs on Windows)

A script to automate the creation of Bengali and English font packages for Magisk and Recovery. Note that the program does not come with any fonts. You have to obtain and provide the Unicode fonts by yourself.

Download and Instructions

You can find the tool and usage instructions on Github.

Already know how to use the tool? Download FontAutomizer-v2.0.1.exe.

Font Freak Installer (runs on Android)

This a font module for Magisk, based on nongthaihoang's CFI. FFI makes the usage simpler and less tiring, taking away the need for endless renaming.

Download and Instructions

You can find the tool and usage instructions on Github.

Already know how to use the tool? Download FFI-v1.02.zip.
